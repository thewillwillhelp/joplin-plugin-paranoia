# Paranoia Joplin Plugin

## Features

Can run simple scripts in next blocks:

    ```js paranoia_script

    ```

    ```paranoia_script

    ```

and show the output in

    ```paranoia_script_output

    ```

## build

For information on how to build or publish the plugin, please see [GENERATOR_DOC.md](./GENERATOR_DOC.md)

icons here: https://fontawesome.com/v5/search

To build run:

> npm run prepare

Also for dev mode joplin allows to set /dist folder for plugin
